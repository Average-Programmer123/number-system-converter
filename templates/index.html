Learn more about the Number System

Number system is the collective term for Decimal, Binary, Octal (OCT), and Hexadecimal (HEX). These are special strings of numbers or letters used as a method of representing numerical values using a set of symbols or digits.

Some special number systems, like Binary, are the only languages that can be understood by a computer.

Binary

The binary system is a numeral system designed to represent numbers using only 0 and 1 as its digits. It is the only language that can be understood by a computer.
Our site helps convert any decimal to binary, but in case you have to do it by hand or don't have internet access, there is a formula for it.
A decimal is any digit from 0–9 and forms the numbers we all know. Since decimal numbers are just powers of 2, we can divide a number by 2 until the quotient is zero.

Here is an example:

Sample number: 13
Calculation:

  • 13 ÷ 2 = 6, remainder 1
  • 6 ÷ 2 = 3, remainder 0
  • 3 ÷ 2 = 1, remainder 1
  • 1 ÷ 2 = 0, remainder 1

Now read the remainders from bottom to top.
Final answer: 1101

Octal

The octal number system is a base-8 numeral system that uses eight digits: 0, 1, 2, 3, 4, 5, 6, 7.

Each digit in an octal number represents a power of 8, just as each digit in the decimal system represents a power of 10.

Example:
Octal number 127₈ equals: 

    1 × 8² + 2 × 8¹ + 7 × 8⁰ 
    = 64 + 16 + 7 
    = 87₁₀

Hexadecimal

The hexadecimal system is base-16 and uses 16 digits: 0–9 and A–F, where A = 10, B = 11, ..., F = 15.

Example: 1A₁₆ = 1×16 + 10 = 26₁₀
